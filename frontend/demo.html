AWS

Amazon Web Services

📝 Exemplo de Prompt

Preciso de uma aplicacao web 
escalavel com:
- 500 usuarios simultaneos
- Banco de dados relacional
- Upload de arquivos
- Cache distribuido
- Monitoramento 24/7

Requisitos:
- Alta disponibilidade (99.99%)
- RTO: 1 hora
- RPO: 15 minutos
- LGPD compliant

🏗️ Arquitetura Gerada

Load Balancer

ALB + Route 53

Compute

ECS Fargate (2-10 tasks)

Database

RDS Aurora PostgreSQL (Multi-AZ)

Cache

ElastiCache Redis (Cluster mode)

Storage

S3 + CloudFront (CDN)

💰 Estimativa de Custos

Compute (ECS Fargate): USD 650/mês

RDS Aurora: USD 400/mês

ElastiCache Redis: USD 200/mês

S3 + CloudFront: USD 150/mês

Load Balancer + Route 53: USD 100/mês

Total Estimado: USD 1.500/mês

🔧 Terraform Module

module "aws_web_app" {
  source = "./modules/web-app"
  
  app_name = "myapp"
  region   = "us-east-1"
  
  ecs_task_count = 3
  container_cpu  = 2048
  container_mem  = 4096
  
  db_instance    = "db.r6i.large"
  cache_node_type = "cache.r6g.xlarge"
}

Google Cloud Platform

GCP - Serviços Gerenciados

📝 Exemplo de Prompt

Plataforma SaaS com:
- 1000 usuarios
- Processamento de dados em tempo real
- Machine Learning (recomendacoes)
- API serverless
- Multiplos formatos (REST, gRPC)

Requisitos:
- Auto-scaling (0-100 instancias)
- Latencia baixa (<100ms)
- Cost-effective
- Logs centralizados

🏗️ Arquitetura Gerada

API Gateway + Load Balance

Cloud Load Balancing + Cloud CDN

Serverless Compute

Cloud Run (auto-scaling)

Database

Cloud SQL + Firestore (NoSQL)

Real-time Events

Cloud Pub/Sub + BigQuery

ML Pipeline

Vertex AI + BigQuery ML

💰 Estimativa de Custos

Cloud Run: USD 400/mês

Cloud SQL: USD 300/mês

Firestore: USD 200/mês

Pub/Sub: USD 100/mês

Vertex AI + BigQuery ML: USD 300/mês

Total Estimado: USD 1.300/mês

🔧 Terraform Module

module "gcp_saas_app" {
  source = "./modules/gcp-saas"
  
  project_id = "my-project"
  region     = "us-central1"
  
  cloud_run = {
    cpu    = 2
    memory = "4Gi"
    max_instances = 100
  }
  
  databases = {
    cloud_sql = true
    firestore = true
  }
}

Microsoft Azure

Azure - Enterprise Cloud

📝 Exemplo de Prompt

Migracao de uma aplicacao 
corporativa para cloud:
- 200 usuarios internos
- Integracao com Active Directory
- Compliance: ISO 27001, GDPR
- Backup e disaster recovery
- Ambiente dev/staging/prod

Requisitos:
- Disponibilidade 99.95%
- Auditoria completa
- Isolamento de rede
- Criptografia end-to-end

🏗️ Arquitetura Gerada

Identity & Access

Azure AD + Conditional Access

Application Gateway

Application Gateway (WAF) + Traffic Manager

Compute

App Service Plan + Container Instances

Data Protection

SQL Database + Key Vault + Backup Vault

Monitoring

Azure Monitor + Sentinel + Policy

💰 Estimativa de Custos

App Service (Premium): USD 350/mês

Azure SQL Database: USD 450/mês

Application Gateway: USD 200/mês

Backup + Storage: USD 250/mês

Monitoring + Security: USD 150/mês

Total Estimado: USD 1.400/mês

🔧 Terraform Module

module "azure_enterprise_app" {
  source = "./modules/azure-enterprise"
  
  resource_group = "rg-production"
  location       = "eastus"
  
  app_service_sku = "P1V2"
  sql_sku         = "S3"
  
  enable_ad_integration = true
  enable_backup         = true
}
